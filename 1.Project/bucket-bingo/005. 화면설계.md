### 벤치마킹
- 

#### 1. 목록
- HOLA처럼, 애플리케이션 첫 진입 페이지를 목록 화면으로 함
- 로고 이쁘게 해야 함!!!
- 화면엔 오프셋 기반으로 가되, 백엔드 상에서는 커서 기반도 가능하도록 개발
#### 2. 상세
- 수정도 지원(수정은 생성 이후)
	- 빙고 칸, 빙고 컬러 수정
#### 3. 생성
- 목록 페이지에서 별도 레이어(ex. 모달)로 생성 
- name / description 등의 메타데이터만을 입력하고, 빙고 칸에 대한 목표 설정은 상세 > 수정 페이지를 활용

### 첫 화면
- 화면엔 오프셋 기반으로 가되 백엔드에선 둘 다 구현
- 조회수: 상세는 포함해야 함
